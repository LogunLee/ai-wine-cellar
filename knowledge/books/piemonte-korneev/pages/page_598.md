# Глоссарий (B → C)

- **Brut** — экстрасухое игристое (классический метод/шампенуа, выдержка на осадке); экспедиционный ликёр **≤14 г/л** после дегоржажа. В Италии метод именуется *metodo classico / tradizionale*.
- **Bussia** — небольшая высококачественная часть зоны Бароло (крю-территория).
- **Ca' (от Casa)** — дом.
- **Cannellino** — полусухой/полусладкий фраскати (от a cannella — «из бочки»).
- **Cannetto** — тростниковая система посадки лоз (юг; «вигвам» из тростника вокруг черенка).
- **Cantina** — погреб(а); **Cantina Sociale** — кооперативное винодельческое предприятие/погреба.
- **Caraffa** — караф/декантер для аэрации вина.
- **Carati** — см. scaraffare (декантация).
- **Caratelli** — небольшие бочонки **50–150 л** (дуб, каштан, фруктовые породы).
- **Casa Vinicola** — виноторговый дом/негоциант (бутилирование из купленного сырья/сусла).
- **Cascina** — ферма/с/х предприятие (обычно указывает на землевладение).
- **Castelli** — замки; **Castello / Castel** — замок.
- **Cerasuolo** — светло-алое, розовое.
- **Cercone** — о неудавшемся вине с неприятным запахом.
- **Champenoise (méthode)** — повторная ферментация в бутылке (≥**16 мес.** совокупно; газ образуется в бутылке, каждая бутылка индивидуальна). В Италии термин выходит из употребления (запрет французских терминов вне Шампани) → *metodo classico / tradizionale*.
- **Charmat** — метод игристых в больших автоклавах (по фамилии Эжена Шармá); массовый стиль, часто для Asti Spumante.
- **Cima** — вершина.
