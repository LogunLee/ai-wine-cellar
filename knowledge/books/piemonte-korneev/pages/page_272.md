# PIEMONTE
**Denominazione di Origine Controllata**

Пьемонте, без преувеличения, самая обширная зона производства в регионе. Она охватывает весь его юг до центра. Это один из новейших DOC в Италии. Региону Пьемонта он понадобился для того, чтобы классифицировать всё то, что не имело официальных классификаций (как известно, в регионе совсем нет местных вин и производится очень немного столового вина, Vino da Tavola).

### Выдержки из технического регламента

Об изменениях и дополнениях в классификации зоны производства см. с. 420.

*\* Декрет Министерства сельского хозяйства: классификация DOC присвоена 30.07.2007. Этот декрет заменил принятый 22.11.1994 (приняты поправки от 13.05.1995, от 22.12.1995, от 2.04.1996 и от 24.08.2001).*

**Основные типы вин, предусмотренные DOC, до 2010 года:**
- Piemonte bianco Cortese // белое сухое
- Piemonte bianco Chardonnay // белое сухое
- Piemonte bianco Moscato // белое сухое
- Piemonte bianco Moscato passito // белое из подвяленного винограда, сладкое
- Piemonte spumante // игристое сухое или сладкое
- Piemonte spumante Pinot bianco // игристое белое сухое
- Piemonte spumante Pinot grigio // игристое белое сухое
- Piemonte spumante Pinot nero // игристое белое сухое
- Piemonte spumante Pinot // игристое белое сухое
- Piemonte spumante Pinot-Chardonnay // игристое белое сухое
- Piemonte spumante Brachetto // игристое красное сладкое или сухое
- Piemonte frizzante Cortese // жемчужное белое сухое
- Piemonte frizzante Chardonnay // жемчужное белое сухое
- Piemonte frizzante Barbera // жемчужное красное сухое
- Piemonte frizzante Bonarda // жемчужное красное сухое
- Piemonte rosso Barbera // красное сухое
