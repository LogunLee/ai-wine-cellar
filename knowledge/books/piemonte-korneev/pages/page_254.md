…**Monferrato Freisa:** DOC Freisa d’Asti.
**Monferrato Dolcetto:** DOC Dolcetto d’Acqui, DOC Dolcetto Ovada, DOC Dolcetto d’Asti.

**Расположение виноградников:** исключены низменности, сырые почвы. В случае использования части винограда с виноградников, относящихся к другим DOC, следует применять самую строгую норму урожайности из норм двух DOC.

**Технические параметры продукта. Минимальное содержание спирта:**

| Тип | Спирт |
|---|---|
| Monferrato rosso | 11% |
| Monferrato bianco | 10% |
| Monferrato Chiaretto / Ciaret | 10,5% |
| Monferrato Dolcetto | 11% |
| Monferrato Freisa | 11% |
| Monferrato Casalese | 10,5% |

**Этикетка.** Только для Monferrato Casalese разрешено использование дополнительных географических указаний. Названия сортов должны быть указаны непосредственно под названием надписью Denominazione di Origine Controllata, которая должна присутствовать непосредственно под надписью Monferrato / Monferrato Casalese.

В 2006 году было подано 2020 заявок на производство с территории площадью 1461 га, итоговый объём произведённого вина — 82 012 гл.

**Лучшие вина зон производства.**
*Barbera del Monferrato (самостоятельный DOC):* Superiore Bricco Battista — Giulio Accornero e Figli.
*Monferrato Rosso:*
- Pin — La Spinetta
