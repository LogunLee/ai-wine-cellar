# Глоссарий (A → B)

- **Anfora / Amphora** — бутылка в форме античной амфоры (характерна для зоны Вердиккьо, Марке).
- **Annata** — год сбора урожая; также базовое («ординарное») вино хозяйства с указанием года (не спец-кюве и не крю).
- **Antico (-a)** — старинный, древний (напр. antica ricetta — старинный рецепт).
- **Apassimento** — подвяливание/заизюмливание винограда после сбора (от apassire).
- **Aperitivo** — аперитив.
- **Arancino** — южный ликёр из апельсиновой цедры (аналог дамского дижестива; ср. лимончелло).
- **Armonico** — гармоничный.
- **Aromi** — ароматы.
- **Asciutto** — очень сухой (чаще об игристых; соответствует extra dry).
- **Asprinio** — приятный терпкий вкус.
- **Assemblaggio** — ассамбляж: смешивание сортов / участков / вин для комплексности или спец-кюве.
- **Astringente** — терпкий; **Astringenza** — терпкость.
- **Azienda** — ферма/хозяйство: **agricola** (с/х угодья, чаще производитель), **vinicola** (виноторговое/негоциант-бутилировщик), **vitivinicola** (винодельческое хозяйство).
- **Barone** — барон / семейство баронов.
- **Barricato / Barricaia / Barricheria** — выдержанное в барриках; погреб под баррики (бочки **225–350 л**).
- **Bianco** — белое (часто маркер ассамбляжа из ≥2 белых сортов).
- **Bicchiere** — бокал.
- **Botte** — бочка/баррель ёмкостью **>10 гл**.
- **Bottiglia** — бутылка.
- **Bricco** — в Пьемонте: верхняя, максимально освещённая часть холма, макушка холма.
