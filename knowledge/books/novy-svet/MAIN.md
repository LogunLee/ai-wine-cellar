# novy-svet — «Вина стран Нового Света» (SWN, 2015) — рабочий файл

## Глобальная цель
Полное распознавание книги в чистый структурированный текст для единой RAG-базы знаний из винных книг. Точность (имена/числа/аппелласьоны), провенанс (книга + страница), сохранение мыслей, переходящих со страницы на страницу.

Это книга 2 из 5 в `knowledge/books/QUEUE.md`. Книга 1 (vina-francii) — ✅ готова.

## Метод
Гибрид: Tesseract (rus+fra, PSM 1) как якорь-страховка от пропусков + полный vision-проход Claude по рендеренным PNG (структура, колонки, цветные врезки, карты) + перекрёстная сверка чисел/имён.

**ВНИМАНИЕ — систематическая OCR-ошибка 3→5.** В vina-francii (тот же издатель/серия/шрифт) Tesseract систематически читал печатную «3» как «5» (даты, площади, проценты). Очень вероятно повторение здесь. ВСЕ числа сверять с PNG; при двусмысленности предпочитать «3» + проверять по фактам/суммам (доли в ассамбляже = 100% и т.п.).

## Формат page_NNN.md
Frontmatter YAML: `book_id, png, printed_page, sections (список), page_kind, starts_mid_thought, ends_mid_thought, ocr, notes`. Тело — чистый текст: склейка переносов; врезки/подписи/сноски/таблицы/карты отдельными блоками `### [врезка]`, `### [подпись]`, `### [сноска]`, `[таблица]`, `[карта]`, `[инфографика]`; спорное/искажённое — реконструировать по смыслу и пометить в `notes`, НЕ выдумывать точные нечитаемые данные.

## РЕЖИМ: автономный гринд
