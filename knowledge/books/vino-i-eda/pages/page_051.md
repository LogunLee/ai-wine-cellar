[таблица] **Вина Пьемонта и сочетания** (наименование → сорт → характеристики → блюда/сочетания)

*Белые вина:*
- **Gavi DOC / Cortese Piemonte DOC** — кортезе — лёгкие белые фруктовые вина с шероховатой кислотностью, пьются молодыми — к аперитиву, салями, рыбе, морепродуктам; форель в кислом соусе, фритто мисто.
- **Roero Arneis DOC** — арнеис — белое вино с лёгкой медовой ноткой — спаржа, кабачки и др. овощи, яйца, неароматные сыры; вителло тоннато, омлет с травами, сыр тома.
- **Chardonnay Piemonte DOC / Langhe Chardonnay DOC** — шардоне — лёгкое и фруктовое при выдержке в стали, тучное и «жирное» при выдержке в дубе — лёгкое шардоне к рыбе, «жирное» к птице; капонет (фаршированные цветы тыквы или кабачка в кляре).
- **Favorita IGT Piemonte** — фаворита — лёгкое, свежее, шероховатое — аперитив, холодные закуски.

*Игристые вина:*
- **Asti Spumante DOCG** — москато — ароматное, игристое, слабоалкогольное (7–8°) — десерты; гриссини, семифреддо.
- **Moscato d’Asti DOC** — москато — игристое, лёгкое, сладкое, слабоалкогольное (5°) — сладости, десерты; торроне, панна котта.
- **Brachetto d’Acqui DOCG** — бракетто — игристое, ароматное, сладкое, фруктовое — фруктовые салаты, десерты; вишня в красном вине.

*Красные вина:*
- **Barbera Vivace (Novello)** — барбера — тёмно-фиолетовое, игристое, фруктовое, пьётся молодым — закуски, салями, ветчина; прошутто.
- **Barbera Piemonte DOC** — барбера — утончённая фруктовость, приятная кислотность — к повседневной еде, ризотто, пасте; ризотто аи фунги (с грибами).
- **Barbera d’Asti DOC / Barbera d’Alba DOC / Barbera del Monferrato DOC** — барбера — сочное, танинное, выразительное, среднетелое — мясная нарезка, блюда на гриле, свинина; ризотто с грибами, боллито мисто, тайарин.
- **Barbaresco DOCG** — неббиоло — элегантное, утончённое, более лёгкое, чем бароло, замечательное для выдержки — кролик, утиное марке, пернатая дичь, тартар, антрекот, блюда с трюфелями, дичь в соусе; стинка ди вителло (телячья голень), карпаччо с трюфелями.
- **Barolo DOC** — неббиоло — мощное, пряное в апогее своей зрелости, замечательное для выдержки — дичь, тартар, антрекот, мощные сыры; бразато аль бароло, капретто арросто (молочный козлёнок), ризотто аль бароло.
- **Roero DOC** — неббиоло, арнеис — ароматное, элегантное, благородное, способное на выдержку, «облегчённая» версия бароло — блюда на гриле, жаркое, сыры; карне круда (сырое рубленое мясо).
- **Nebbiolo d’Alba DOC** — неббиоло — паста с соусом, эскалоп, жаркое из телятины; тайарин аи фегателли (лапша с печёночным соусом).
- **Dolcetto DOC** — дольчетто — ягодное, мягкое, приятное, кислотное — простые блюда, холодные закуски; банья кауда.
- **Dolcetto Superiore (d’Alba, d’Asti)** — дольчетто — с приятной горечью, кислинкой, вишнёвым тоном — мясное, жаркое из свинины; финандзьера.
- **Freisa d’Asti DOC / Freisa di Chieri DOC / Colli Tortonese DOC** — фрейза — красная вишня, ягодные тона, лёгкая кислинка — закуски, паста, белое мясо; рагу из птицы.
- **Grignolino del Monferrato Casalese DOC / Grignolino d’Asti DOC** — гриньолино — розовое, лёгкое, с приятной горчинкой — закуски, рыба, птица; фондута (фондю), раклет.

*Крепкие вина:*
- **Moscato Passito DOC** — москато — насыщенное, ароматное, с выраженным тоном муската — бисквиты, сладости, шоколад; бискотти кон ноччоле (бисквиты с орехами).
- **Barolo Chinato** — неббиоло — крепкое (17°), настоянное на ароматных травах — дижестив; Vin Brulé (глинтвейн).
