[подзаголовок] Бокалы для шампанского и игристых вин
[нарратив] Среди разнообразных бокалов для игристых вин наиболее очевидна «классовое расслоение». Так называемые кремины, которые, по легенде, некогда были «креманы» по форме груди большой любительницы шампанского Марии-Антуанетты из исторической пирамиды, сегодня всё ещё производятся, но рекомендуются им можно разве что под полусладкие игристые вина, сделанные по методу шарма. Универсальной формой «шампанского» бокала считают «флейт», но профи нередко наливают такой бокал «нелинейным» так, чтобы наполнить и не сделать столб, приходится запахивать голову упрощённо. Для недорогих игристых вин для игристых высокого класса и для игристых высшего качества можно выбирать всю эту целинейную форму. Хороший бокал — то, что остаётся выгнутым, должен всё же заметно расширяться кверху. У него также длинная ножка — точечное углубление на донышке чаши (благодаря ему перлин будет более тонким и устойчивым).

[фото] Spiegelau Willsberger Anniversary — бокал для шампанского; Josephine — бокал для шампанского.

[подзаголовок] Бокалы для креплёных и десертных вин
[нарратив] Креплёное вино можно пить из стандартных бокалов для красных и белых вин (для порта — бокал типа «Бордо», для хереса — «белый» бокал). Но для особенных образцов можно приберечь особенные бокалы.

[подкарточка] Порт
[нарратив] Бокал для винтажного портвейна напоминает по очертанию «бордоский», но горазда меньше его по объёму. И хотя для бокалов ручной работы — точечно подчёркивает аромат вина (тон, специи, трюфели, кофе), горлышко порт довольно крепкое; важно, чтобы тонкие нюансы букета раскрывались на фоне вина «спиртуозности».

[подкарточка] Херес
[нарратив] При разработке формы бокала для хереса за ориентир взяли традиционный бокал, который был довольно маленьким (чтобы охлаждённый вино подавалось самим маленькими порциями). Пить херес следует холодным, а порция совсем небольшая (чтобы не успело нагреться).

[фото] Spiegelau Grand Palais Exquisit — бокал для портвейна; Spiegelau Adina — бокал для хереса.
