## Указатель апелласьонов и стилей (B–M)
Brouilly → гамэ; Brunello di Montalcino → санджовезе; burgundy красная → пино нуар; burgundy белая → шардоне / алиготе; Cahors → мальбек; Carmignano → санджовезе; cava; Chablis → шардоне; Châteauneuf-du-Pape → гренаш нуар / мурведр; Chianti → санджовезе; Chinon → каберне фран / шенен блан; Chiroubles → гамэ; claret → каберне совиньон / мерло / Pomerol; Condrieu → вионье; Cornas / Côte Rôtie / Crozes-Hermitage → сира; Côtes du Rhône → гренаш / сира / Rhône-белые бленды; fino → херес; Fitou → красные Languedoc-Roussillon; Fleurie → гамэ; Gavi → кортезе; Gigondas → гренаш; Graves белое → бленды совиньон-семильон (сухие); Hermitage → сира / Rhône-белые бленды; Juliénas → гамэ; Lambrusco; Mâcon → шардоне / гамэ / пино нуар; Madeira; Madiran → таннат; manzanilla → херес.
