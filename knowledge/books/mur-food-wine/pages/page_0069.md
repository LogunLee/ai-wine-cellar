## Указатель апелласьонов и стилей (M–S)
Marcillac → фер серваду; Médoc → каберне совиньон; Menetou-Salon → совиньон блан; Monbazillac → Sauternes; Montlouis-sur-Loire → шенен блан; Montsant → Priorat; Morgon → гамэ; Moscato → мускат; Moulin-à-Vent → гамэ; Muscadet; oloroso / palo cortado / palomino / pedro ximénez → херес; Pessac-Léognan → каберне совиньон / бленды совиньон-семильон (сухие); Pomerol; Priorat; Prosecco; Quincy / Reuilly → совиньон блан; Rhône-белые бленды; Ribera del Duero → темпранильо; Rioja → темпранильо / грасиано; Rosso di Montalcino / Rosso di Montepulciano → санджовезе; St Amour → гамэ; St Emilion → мерло; St Joseph → сира; St Nicolas de Bourgueil → каберне фран; Sancerre → совиньон блан; Saumur (blanc) → шенен блан; Saumur (rouge) / Saumur-Champigny → каберне фран.
