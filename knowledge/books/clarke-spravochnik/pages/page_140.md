[нарратив] **Стили.** Секрет успеха — старые виноградники + современные методы: питкие красные (клубника, мелкие сливы, травы), сухие дерзкие розовые, лёгкие цитрусовые белые. Серьёзные стили тоже есть — пряных согревающих, чуть ванильных от дуба вин делают чуть ли не больше прежнего; есть сливочные дубовые белые; стиль великого хереса не меняется поколениями.

[нарратив] **Кава.** Всё испанское игристое по технологии вторичной ферментации в бутылке (как в Шампани) называется **Кава**; почти вся — из **Пенедеса** (окрестности Барселоны). Раньше узнаваемо-простая, но дешёвая; теперь с развитием технологий характерны свежесть, чистота, лимонные тона. Сорта — нейтральные «парельяда», «макабео», «шарельо», иногда + «шардоне» для сливочности.

[врезка] **Испанская классификация.**
- **DOCa** (Denominación de Origen Calificada) — высшая категория, только для вин с многолетней традицией; на данный момент — **только Риоха и Приорат**.
- **DO/DOP** (Denominación de Origen / Protegida) — стандартная категория качественных вин с довольно строгими требованиями.
- **Vino de pago** — для отдельных производителей/виноградников уровня DO (около дюжины).
- **IGP** (местные вина, с географическим указанием) — типичные для региона; многие претендуют на DO.
- **Vino de mesa** (столовые) — самые простые, могут обозначаться просто «вино».

[подпись] Из Барселоны вглубь региона Приорат — маленькие террасные виноградники среди миндаля и оливок на фоне гор.
