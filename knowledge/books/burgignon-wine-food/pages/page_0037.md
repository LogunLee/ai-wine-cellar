[нарратив] **Пойяк (Pauillac), Бордо.** Великий апелласьон левого берега Жиронды; лучше выдержать ~10 лет. Молодой слишком доминирует (дубовый букет: чёрная смородина, ежевика, черника, белый табак). Зрелый — кожа, кедр, иногда эвкалипт; баланс для высокой кухни.

Сочетания:
- Ягнятина, прежде всего седло барашка (с корочкой из хлебных крошек); гарнир пом-буланжер (не зелёная фасоль — даёт твёрдость вину).
- Зобная железа телёнка в лёгком сливочном соусе (шелковистые текстуры).
- «Дружит» с трюфелем (как сен-жюльен, сент-эстеф, марго), хотя с трюфелем лучше правый берег (помероль, сент-эмильон).
- Воспоминание: Château Léoville-Las-Cases 1982 к телячьим ножкам по Алену Дюкассу; зрелый сен-жюльен с жареной телятиной/говяжьим стейком и гратеном из макарон.
- Тушёные блюда (наварен из ягнёнка) — придаёт плотность.
- Пернатая дичь (жареный бекас) предпочтительнее дичи с шерстью.
- Сыры: горные, выдержанные голландские (эдам, мимолетт, гауда); лучшее — сен-нектер (ни вино, ни сыр не доминируют).

[врезка] **Вина, близкие по стилю:** гран крю Медока; сент-эстеф; сен-жюльен; марго; листрак; Каберне Совиньон (США, Австралия, Чили, Италия).

[врезка] **Блюда, подчёркивающие достоинства:** жареная одним куском телятина с гратеном из макарон; седло ягнёнка; жареный бекас; голландские сыры; сыр сен-нектер.
