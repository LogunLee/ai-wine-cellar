[нарратив] **Кроз-Эрмитаж (Crozes-Hermitage), Север Долины Роны.** Для автора — вино, ассоциируемое с кровью (детское воспоминание о Дне Сен-Кошона). Мясистое, многообещающее; насыщенный цвет с фиолетовыми бликами; ароматы цветущего ириса, маков, чёрного перца, слегка дикий характер.

Сочетания:
- Лучшая пара — кровяная колбаса (деревенская, как у мадам Люлю в ресторане L'Assiette); креольские колбасы со специями. Региональный сангет (sanguette — птичья кровь с петрушкой и чесноком, обжаренная галетой).
- Молодой и прохладный — паштеты из дичи, фазана, зайца; алиго из Аверона с тулузскими сосисками; филе миньон из свинины с перцем, свиная шейка на гриле.
- Мощная рыба: барабулька, морской ёж (йодистый вкус) — можно отказаться от белого. Ризотто с чернилами каракатицы (по Алену Дюкассу) — насыщенность вина совпадает с «чернильным» блюдом.
- Соблазнительное вино для пикника, когда хочется пить.

[врезка] **Вина, близкие по стилю:** сен-жозеф красное; вина из сорта Мондёз (Mondeuse) из Савойи.

[врезка] **Блюда, подчёркивающие достоинства:** паштеты из дичи; чёрная кровяная колбаса с жареными яблоками; филе миньон из свинины с зелёным перцем; ризотто с чернилами каракатицы.
