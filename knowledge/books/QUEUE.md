## Процедура запуска НОВОЙ книги (когда предыдущая = ✅ готова)

Рабочая папка интермедиатов: `C:\Users\LoGun\AppData\Local\Temp\pdf_ocr_<book_id>\` (pages\, tess\). tessdata переиспользуем из `...\pdf_ocr_test\tessdata` (rus/fra/eng/osd; для Пьемонта доскачать `ita.traineddata` из tessdata_best).

1. **Рендер** всех страниц в PNG 300dpi (PyMuPDF) → `pages\page_NNN.png`. Скрипт-шаблон — как в `vina-francii` (см. её MAIN.md), поменять `src` и `out`. Запускать в фоне.
2. **Tesseract-якорь** по всем страницам → `tess\page_NNN.txt` (для Пьемонта `-l rus+ita`, остальные `-l rus+fra`, PSM 1). В фоне.
3. Создать `knowledge/books/<book_id>/meta.yaml` (по выходным данным книги) и `MAIN.md` (статус, формат, журнал) — по образцу vina-francii.
4. OCR батчами ~5-8 стр.: читать PNG+якорь, писать `pages/page_NNN.md`, вести журнал в MAIN.md.
5. По завершении книги: собрать `book.md` (склейка по флагам ends/starts_mid_thought) и `chunks.jsonl`, отметить книгу ✅ здесь, перейти к следующей.
