# 50 фактов про игристые вина (начало)

- **Méthode traditionnelle** (классический/традиционный метод) — вторичная ферментация в закупоренных бутылках. Называть его «шампанским методом» вне Шампани теперь запрещено.
- **Champagne AOC** — шампанское из Шампани, самое вожделенное; ничто другое шампанским называться не может. Это предмет роскоши: дорого, достать непросто, нужен достойный повод.
- **Винтажное (миллезимное)** — шампанское одного года урожая (год на этикетке) считается особым; «декларировать винтаж» можно только в выдающиеся по погоде годы, что строго контролируют власти региона Шампань.
- **Ассамбляж:** по старинной традиции базовое шампанское — смесь вин разных лет (в ассамбляж могут входить 10, 50 и даже 500 отдельных вин-составляющих с разных крю, по-разному винифицированных).
- **Основные категории шампанского:** невинтажное (базовый брют дома); винтажное (миллезимное); престижное кюве (Cuvée de Prestige). Все три могут быть розовыми (Rosé).
- **Альтернативы по классическому методу:** игристые других регионов Франции (креманы, бланкетты); кава (Испания, преим. Каталония); британские игристые; южноафриканские Cap Classique; авторские игристые виноделен по всему миру.
- **Сорта:** используются и белые, и красные (с белым соком, красной шкуркой). Только из белых → **Blanc de Blancs**; только из красных → **Blanc de Noirs**.
