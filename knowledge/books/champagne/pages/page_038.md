# Словарик терминов (обязательное)
- **Перляж (perlage)** — игра пузырьков в бокале.
- **Ассамбляж (assemblage)** — смешивание разных исходных тихих вин.
- **Винтаж (миллезим)** — год урожая; для немиллезимных используют аббревиатуру **NM**.
- **Первичная винификация** — превращение сока в вино (спиртовое брожение и работа с суслом).
- **Фудры / буты** — большие старые дубовые бочки (в Шампани).
- **Ремюаж (remuage)** — переворачивание бутылок горлышком вниз для сведения дрожжевого осадка; бывает ручным (пюпитры) и автоматическим (жиропалеты).
- **Сабраж (sabrage)** — открывание бутылки твёрдым предметом с ровной гранью (сабля и т.п.).
- **Дегоржаж (dégorgeage)** — удаление осадка «выстреливанием».
- **Льё-ди (lieu-dit)** — небольшой именованный участок виноградника без классификации, но с характерным терруаром.
- **Терруар (terroir)** — совокупность свойств почв и микроклимата участка + адаптированный клон лозы.
- **AOC** — французский аппелласьон (наименование, контролируемое по происхождению).
- **Негоциант** — производитель, покупающий виноград, делающий и продающий вино (с маркетинговыми силами).
- **Рекольтан (récoltant)** — мелкий независимый виноградарь, делающий «авторское» шампанское в малых объёмах.
- **Шеф-де-кав (chef de cave)** — глава погреба, главный энолог дома (термин именно для Шампани; в других регионах — maître de chais).
- **Мениль-сюр-Ожер (Le Mesnil-sur-Oger)** — коммуна в Кот-де-Блане, лучшее гран крю для шардоне.
- **Блан-де-блан** — игристое только из белого сорта (для шампанского — 100% шардоне).
- **Cuvée de Prestige** — топовая марка дома.
- **Креман** — хорошие французские игристые не из Шампани (в наименовании обязательно Crémant AOC).
- **Зект (Sekt)** — немецкие игристые. **Cava DO** — лучшие игристые Испании. **Cap Classique** — игристая гордость ЮАР.
