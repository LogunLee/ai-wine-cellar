# Кашрут в пузырьках (кошерное игристое)
- Игристые (в т.ч. кошерные) делают в Израиле — например **Golden Heights / Yarden Blanc de Blancs Brut** (Голанские высоты). Кошерные кюве есть не только в Израиле: у **Laurent-Perrier** и **Drappier**; есть кошерные кавы и просекко.
- Правила те же, что для тихого кошерного: нельзя использовать урожай виноградника в его первые **4 года** и каждый **7-й год**; все операции, где человек контактирует с вином, должны выполнять евреи, «соблюдающие субботу»; для осветления запрещены вещества животного происхождения. Метод (классический или шарма) значения не имеет.

[таймлайн] Среди исторических вех: **Gosset** — старейший из существующих домов Шампани (изначально делали тихие вина). *(Год основания на ленте таймлайна OCR исказил — сверить по PNG.)*
